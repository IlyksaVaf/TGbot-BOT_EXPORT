# TGbot
Проект изначально чисто личная задумка, для личного пользования.

Суть бота в том, что ему присылаешь ссылки, он их хранит в базе данных, по требыванию выдает. 
Есть возможность добавлять названия ссылкам.
Есть возможность удалять ссылки выборочно.
Есть возможность удалять дубликаты ссылок, и оставлять только первую копию.
Есть возможность удалить вообще все ссылки сразу.
Есть меню /help с полезной информацией для пользователя.

В качестве базы данных используется PostgreSQL:
3 таблицы:
1. chats (chat_id, message_id)
2. links (chat_id(fk), links_text, id)
3. links_name (links_name, links_id(fk))
