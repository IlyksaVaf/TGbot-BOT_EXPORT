# TGbot
Проект изначально чисто личная задумка, для личного пользования.

Юзернейм бота в телеграмме - @MyIsbrannoeBot

Суть бота в том, что ему присылаешь ссылки, он их хранит в базе данных, по требыванию выдает. 

Возможности бота:
1.  Возможность добавлять названия ссылкам.
2.  Возможность удалять ссылки выборочно.
3.  Возможность удалять дубликаты ссылок, и оставлять только первую копию.
4.  Возможность удалить вообще все ссылки сразу.
5.  Меню /help с полезной информацией для пользователя.


В качестве базы данных используется PostgreSQL:

3 таблицы:
1. chats (chat_id, message_id)
2. links (chat_id(fk), links_text, id)
3. links_name (links_name, links_id(fk))
